## 5. ‚òÅÔ∏è Cloud Security :

## 6. üíª Endpoint Security :

<table> <tr> <td><strong>Threats</strong></td> <td>Malware, Ransomware, Unauthorized device access</td> </tr> <tr> <td><strong>Tools & Techniques</strong></td> <td>Antivirus, EDR (Endpoint Detection & Response), Device Hardening</td> </tr> <tr> <td><strong>Goal</strong></td> <td>Ensure every device connected to the network remains secure</td> </tr> <tr> <td><strong>Analogy</strong></td> <td>Saftey</td> </tr> </table>
